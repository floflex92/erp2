# Agent « Task Manager »

name: task-manager-helper
summary: Assistant local pour le développement du module de gestion des tâches.
inputs:
  - name: task-title
    type: string
    description: Titre de la tâche
  - name: action
    type: string
    description: Action à exécuter (add/delete/toggle)

commands:
  - id: addTask
    description: Ajouter une tâche
  - id: deleteTask
    description: Supprimer une tâche
  - id: toggleTask
    description: Marquer une tâche comme complétée/décomplétée
  - id: listTasks
    description: Afficher les tâches actuelles (titre + état)
  - id: clearTasks
    description: Supprimer toutes les tâches (reset localStorage)
