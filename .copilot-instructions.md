# Copilot Custom Instructions

## À propos de moi
Je suis développeur travaillant sur une application React + Vite + TypeScript, spécialisée dans la gestion de flotte. Je veux un code propre, maintenable et en français, avec un minimum de dépendances externes.

## Comment Copilot peut m’aider
- proposer des complétions et des refactorings en respectant la structure existante du projet
- générer des composants React fonctionnels TypeScript et des pages accessibles
- compléter les fichiers de configuration et de documentation (AGENTS.md, instruction Copilot)
- éviter les changements non sollicités hors périmètre

## Mode d’emploi pour Codex/Claude
1. Lire intégralement `AGENTS.md` et `PROMPT_BASE_IA.md` avant toute émulation d’action.
2. Toujours répondre en français.
3. Ne pas exécuter d’actions destructrices (git reset --hard, push --force, netlify deploy) sans confirmation explicite.
4. Proposer d’abord un mini plan en 3 à 4 étapes.
5. Ne modifier que les fichiers nécessaires pour la demande.
6. Fournir un rapport final avec : fichiers modifiés, commandes run, validations réussies, points non faits.
